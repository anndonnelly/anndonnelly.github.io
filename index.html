Speakeasy

Discover, share and review great drinks and distilleries

Created by:

Ann Donnelly

Skills:

Python, Flask, React, Redux, PostgreSQL, SQLAlchemy, Alembic, Docker)

Countable

A social media app to hold politicians accountable

Created by:

Ann Donnelly

Skills:

JavaScript, Express, React, Redux, PostgreSQL, Sequelize, Node.js HTML5, CSS, AWS

Countable is a fullstack app for users to voice concerns about issues in their neighborhood and to hold politicians accountable in an attempt to make real change. Formik forms with yup validations are an example of an open source library I included in this project. In this project I made expensive backend queries and as the current database is not extensive it works for this project but sequelize pagination is something I would consider for a larger scale project down the line.
