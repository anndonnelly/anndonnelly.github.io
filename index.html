A task management web app modeled after Remember the Milk

Created by:

Ann Donnelly

Skills:

Remember the CRUD is a task management app, designed to allow amateur and professional programmers to better organize their programming projects. I collaborated with three other engineers using a kanban board to divide features and goals for efficient project completion. I was the scrum master for this project. We devised a system for the simplified manipulation of the HTML DOM.
